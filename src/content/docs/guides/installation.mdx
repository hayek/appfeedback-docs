<Aside title="Release status">
The code for all three platforms is public on GitHub. Formal registry releases are rolling out:
- **Apple** — released: add the SwiftPM package at `from: "0.1.0"`.
- **Web** — `@appfeedback/*` packages build and pass their suites; **npm publishing is in progress**. During early access, install from the [`appfeedback-web`](https://github.com/hayek/appfeedback-web) workspace.
- **Android** — the shared core (transports + `FeedbackClient`) and the Compose UI module (`FeedbackSheet`) ship and are conformance-tested; **Maven Central publishing is in progress**. During early access, build from the [`appfeedback-android`](https://github.com/hayek/appfeedback-android) repo.
</Aside>

<Tabs syncKey="lang">
<TabItem label="Apple (Swift)">

**Swift Package Manager** — add to `Package.swift`:

```swift
.package(url: "https://github.com/hayek/AppFeedbackSDK", from: "0.1.0"),
```

Then depend on the products you need:

```swift
.target(name: "YourApp", dependencies: [
  .product(name: "AppFeedbackCore", package: "AppFeedbackSDK"),
  .product(name: "AppFeedbackUI", package: "AppFeedbackSDK"), // optional drop-in sheet
])
```

Or in Xcode: **File → Add Package Dependencies…** and paste the repo URL. Platforms: iOS 17 · macOS 14 · watchOS 10 · tvOS 17 · visionOS 1.

</TabItem>
<TabItem label="Web (TypeScript)">

```bash
npm i @appfeedback/core @appfeedback/widget
# React component:
npm i @appfeedback/react
# Server-side relay handler:
npm i @appfeedback/relay
```

- `@appfeedback/core` — framework-agnostic: wire format + `RelayTransport`/`DirectGitHubTransport`.
- `@appfeedback/widget` — the drop-in feedback form.
- `@appfeedback/react` — `<FeedbackForm>` wrapper.
- `@appfeedback/relay` — the handler you deploy ([guide](../relay/)).

ESM, dual-typed; React is a peer dependency (`>=18 <20`).

</TabItem>
<TabItem label="Android (Kotlin)">

Maven Central coordinates (publishing in progress):

```kotlin
dependencies {
  implementation("io.github.hayek:appfeedback-android:0.1.0")
  implementation("io.github.hayek:appfeedback-android-compose:0.1.0") // drop-in FeedbackSheet
}
```

minSdk 24. Until the release lands, build from the [`appfeedback-android`](https://github.com/hayek/appfeedback-android) repo.

</TabItem>
